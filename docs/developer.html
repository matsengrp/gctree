Open source code repository

All code is freely available at https://github.com/matsengrp/gctree

Developer tools

Developer install:

make install

Run tests:

make test

Format code:

make format

Lint:

make lint

Build docs locally (you can then see the generated documentation in docsrc/_build/html/index.html.):

make docs

Deploy docs to docs/ directory:

make deploy
